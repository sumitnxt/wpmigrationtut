Chances are that you will be developing on a localhost ( local machine ). We prefer this because localhost is fast. And speed is a very important factor during development. But it has one disadvantage. Remote server has softalicious so its very easy to make clone sites. Therefore we need this tutorial because one of the challenges is migrating a wordpress site from localhost to remote server. Lets make a list of all types of migrations. 1. localhost same pc to localhost same pc 2. localhost one pc to localhost another pc 3. localhost to remote server 4. remote same host to remote same host ( softalicous clone works 100% ) 5. remote one host to remote second host ( sometime softalicious clone works but not safe) 6. remote server to localhost.
